# CLI Help (git / gh)

Goal: show the official CLI help for a command.

These commands are read-only.

## git help

```bash
git help
git help -a
git help <command>
git <command> -h
```

Examples:

```bash
git help worktree
git help tag
git worktree -h
```

## gh help (optional)

If `gh` is installed:

```bash
gh help
gh help release
gh release create --help
```
