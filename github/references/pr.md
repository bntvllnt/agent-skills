# Pull Requests (gh pr)

List PRs:

```bash
gh pr list
```

View PR:

```bash
gh pr view <number>
gh pr view <number> --comments
gh pr view <number> --web
```

Create PR:

```bash
gh pr create
```

Checks:

```bash
gh pr checks
```

Review / comment:

```bash
gh pr review <number> --comment --body "<comment>"
```

Merge (confirm first):

```bash
gh pr merge <number>
```
