# Issues (gh issue)

List issues:

```bash
gh issue list
```

View an issue:

```bash
gh issue view <number>
gh issue view <number> --web
```

Create:

```bash
gh issue create
```

Comment:

```bash
gh issue comment <number> --body "<comment>"
```

Close / reopen (confirm first):

```bash
gh issue close <number>
gh issue reopen <number>
```
