# Repos (gh repo)

Common repo operations.

Read-only:

```bash
gh repo view
gh repo view --web
```

Clone:

```bash
gh repo clone OWNER/REPO
```

Fork:

```bash
gh repo fork OWNER/REPO
```

Create:

```bash
gh repo create
```

Sync fork:

```bash
gh repo sync
```

Note: for operations that change repo settings (rename, delete, archive), confirm first.
